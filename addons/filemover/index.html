Addons - File Mover

Overview –  Client Tools –  Archive Search –  File Mover –  YarraLink

 Introduction

Yarra FileMover is a small helper tool that moves files regularly from a source location to an archive destination. If the file transfer fails, it sends out email notifications to specified recipients.

We are using it to move processed raw-data files from our YarraServers to a central archive location on a network share. Because the network transfer takes time, we decided to decouple the file transfer from the YarraServer process, so that the server process doesn’t get blocked due to the transfer operation.
